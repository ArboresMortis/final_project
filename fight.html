Cursive is kinda dumb.

In the modern day, there is no use for cursive writing in the average life. It's simply harder to read, harder to write, and makes it more difficult to transfer information to a digital format.

Public schools have started to take the sensible step of phasing out teaching cursive, because allmost all modern writings are done in plain text, so they are actually legible without having to learn a second alphabet.

Using plaintext is also much easier for a computer to do, because it doesn't have to deal with extra rules on how to connect things together. They already have enough trouble with languages like arabic, lets give them a break.

Furthermore, the extra time taken to teach cursive could be better spent on doing things like math, or teaching kids reading conprehention, or science, or a million other more useful topics that aren't cursive.

Being a language prescriptivist compleatly misses the point of language.

Language has to change over time, or it would become unusable. We need new vocabulary to refer to new things, or we need to change the definition of old vocab to include the new things, or else everyone will be forced to use long sentences to refer to relativly simple concepts.

This idea of a language purist is especially silly for the english language, because there are practically no rules to uphold. Ending sentences with prepositions is an ok thing to do. Grammer understood it will be, if you mix up the order.
